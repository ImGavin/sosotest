Jenkins插件介绍

1. 插件在哪里找？

1)项目仓库代码里 AutotestWebD/sosotest-jenkins-plugin/sosotest.hpi

2. 插件怎么安装？

1)Jenkins管理员身份，进入 Manage Jenkins -> System Configuration -> Manage Plugins
2)在 Advanced -> Upload Plugin 选择 sosotest.hpi ，然后点击 Upload 上传，上传完成即可使用

3. 插件如何使用？

1)在构建部分增加构建步骤 sosotest execute plugin

2)配置sosotest任务执行需要的信息
